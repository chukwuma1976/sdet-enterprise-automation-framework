# Performance Tests

Run smoke test:
```
k6 run performance-tests/k6/smoke/users-smoke.js
```
Run load test:
```
k6 run performance-tests/k6/load/users-load.js
```
Run spike test:
```
k6 run performance-tests/k6/spike/users-spike.js
```
Run all performance tests
```
./scripts/run-performance-tests.sh
```
